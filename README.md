## 🚀 Featured Projects

### 🌱 Soil Fertility Analyzer Using Machine Learning
**IoT-based soil analysis system that predicts soil fertility and recommends crops using sensor data and ML**
- Collects soil parameters such as **NPK values, moisture, and environmental conditions** using sensors
- Machine learning model predicts **soil fertility level and suitable crops**
- Real-time visualization through a web interface
- Helps improve agricultural decision-making using data-driven insights

**Tech Stack:** `Python` `Scikit-learn` `NumPy` `Pandas` `IoT` `Sensors`

### 🪨 Edge-Based Landslide Detection System
**Distributed IoT system for early landslide detection using environmental sensors and edge computing**
- Detects **ground movement using gyroscope sensors**
- Monitors rainfall intensity using **rain sensors**
- Uses **edge nodes and end nodes architecture** for localized processing
- Edge computing reduces latency and enables faster hazard alerts
- Designed for deployment in landslide-prone regions

**Tech Stack:** `Embedded C` `ESP32` `IoT` `Edge Computing` `Microcontrollers`

🔗 [View Project](#)

---

### ✋ Hand Gesture to Speech System
**Assistive embedded system that converts hand gestures into speech output**
- Detects hand gestures using **sensor-based input**
- Converts gestures into **text and speech output**
- Bluetooth module sends output to an **Android phone**
- Output displayed using **Serial Bluetooth Terminal application**

**Tech Stack:** `Embedded C` `Arduino` `Sensors` `Bluetooth Communication`
